# Panda Blog Module for PrestaShop 8+

A professional, lightweight, and SEO-friendly blog engine designed specifically for **PrestaShop 8**. This module allows you to manage articles, categories, and dynamic content with ease.

> [!IMPORTANT]
> **Code Preview Only**: This repository is for portfolio and code review purposes. All rights reserved. No license is granted for public or commercial use.

## 🚀 Features

- **Modern Architecture**: Built using Symfony components and PrestaShop 8 standards.
- **Media Management**: Easy image uploads for thumbnails and post content.
- **Clean UI**: Integrated with the PrestaShop Back-Office design system.
- **Hooks Ready**: Display recent posts or categories anywhere in your store.

## 🛠 Tech Stack

- **Backend**: PHP 8.1+, Symfony
- **Frontend**: Twig, JavaScript (ES6), CSS3, SASS
- **Database**: MySQL (PrestaShop DB Engine)

## 📖 Project Overview

This module was developed to demonstrate clean code practices in the PrestaShop ecosystem. It follows the **MVC pattern** and utilizes PrestaShop's modern ObjectModel and Controller structures.

### Key Implementation Details:
- Custom database tables for posts and categories.
- Integration with PrestaShop's native Image Manager.
- Custom hooks for front-end display.
